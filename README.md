# Retail-sales-analysis-dashboard-
A comprehensive Retail Insights Dashboard designed to analyze and visualize retail sales performance across different products, regions, and time periods. This project helps in understanding customer buying patterns, tracking revenue trends, identifying top-performing products, and improving data-driven decision-making for better business growth
Description: This project analyzes retail sales data to identify trends and patterns using Power BI.
Tools Used:
Power BI
Dataset:
Superstore Sales Dataset
